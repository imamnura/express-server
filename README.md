# Express-server

Express server is a backend service with express js framework

## Instalation

Use the package manager npm or yarn
```bash
yarn
````

````bash
npm i
````

## Usage

1. create file .env to run

````bash
ADMINER_PORT=4000
DB_PORT=5432
DATABASE_URL=""
DB_USER=""
DB_PASSWORD=""
````

3. run code

```bash
npx nodemon .
````
